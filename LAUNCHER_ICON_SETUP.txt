=====================================
LAUNCHER ICON SETUP - KriptoMeter
=====================================

Untuk mengganti icon aplikasi dengan logo custom Anda:

LANGKAH 1: Siapkan Logo PNG
-----------------------------
- Format: PNG dengan transparent background
- Size: 512x512 px (recommended) atau minimal 192x192 px
- Nama file: Bebas (akan di-generate ke berbagai size)

LANGKAH 2: Generate Icon di Android Studio
-------------------------------------------
1. Buka Android Studio
2. Klik kanan folder: app/src/main/res
3. Pilih: New → Image Asset
4. Konfigurasi:
   - Asset Type: Image
   - Path: [Pilih logo PNG Anda]
   - Trim: Yes (untuk remove padding otomatis)
   - Resize: 100%
   - Shape: Circle/Squircle/None (pilih sesuai design)
   - Background Color: #1A1D2E atau #14AFAF (cyan)
5. Klik Next → Finish

LANGKAH 3: Build & Install
----------------------------
Build → Clean Project
Build → Rebuild Project
Run aplikasi

Icon baru akan tampil di launcher!

=====================================

ATAU MANUAL (Advanced):
=====================================

Letakkan file PNG di folder-folder ini dengan nama yang sesuai:

app/src/main/res/mipmap-mdpi/

  - ic_launcher_round.png (48x48 px)

app/src/main/res/mipmap-xxxhdpi/
