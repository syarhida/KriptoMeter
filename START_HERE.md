   - Bold, 18sp, warna putih
   - Contoh: Bitcoin, Ethereum, Tether

   - 12sp, warna abu-abu (#B0BEC5)
   - Contoh: BTC, ETH, USDT

4. **Harga USD** (kanan)
   - Bold, 20sp, warna cyan (#14AFAF)
   - Format: $XX,XXX.XX
   - Update real-time dari API
